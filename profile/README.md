# Snaplink / Revede

스냅링크는 고객과 스냅 작가를 연결하는 C2C 스냅 촬영 중개 플랫폼입니다.  
React Native 기반 iOS·Android 앱, React 기반 랜딩페이지, 관리자 페이지, AI 이미지 보정 실험을 중심으로 제품 출시와 운영 검증을 진행했습니다.

## Repositories

| Repository | Description |
|---|---|
| [SNAPLINK_FRONT](https://github.com/REVEDE-SNAPLINK/SNAPLINK_FRONT) | React Native 기반 iOS·Android 모바일 앱 |
| [SNAPLINK_WEB](https://github.com/REVEDE-SNAPLINK/SNAPLINK_WEB) | 서비스 소개, FAQ, 공지, 문의를 위한 React 랜딩페이지 |
| [SNAPLINK_ADMIN](https://github.com/REVEDE-SNAPLINK/SNAPLINK_ADMIN) | 예약, 사용자, CS, 마케팅, 운영 데이터 확인을 위한 관리자 페이지 |
| [SNAPLINK_IMAGE_AGENT](https://github.com/REVEDE-SNAPLINK/SNAPLINK_IMAGE_AGENT) | 스냅 이미지 자동 보정 실험을 위한 AI agent |
| [SNAPLINK_BACK](https://github.com/REVEDE-SNAPLINK/SNAPLINK_BACK) | Backend repository |

## Product Scope

- 스냅 촬영 카테고리 탐색
- 작가 검색 및 프로필 조회
- 작가 포트폴리오 확인
- 문의·예약 플로우
- 후기 작성 플로우
- 사용자 회원가입·로그인 및 상태 관리
- 랜딩페이지 기반 사용자 유입
- 관리자 페이지 기반 운영 관리
- Firebase Analytics, Airbridge, BigQuery 기반 사용자 행동·유입 데이터 계측

## Current Status

- 2026년 2월 iOS·Android 앱 출시
- 초기 사용자 약 120명 가입
- 본격적인 마케팅 및 PMF 검증은 진행 전 단계
- 초기 MVP 출시와 사용자 행동 데이터 수집 환경 구축 완료
